# PeakOS Brand Identity

## Core Values
*   **Fluidity**: Animations should be 60fps+ and physics-based.
*   **Clarity**: Glassmorphism (blur) usage to indicate depth.
*   **Simplicity**: Hide complexity until needed.

## Typography
*   **Primary Font**: Inter (closest open alternative to SF Pro).
*   **Monospace**: JetBrains Mono.

## Color Palette
*   **Peak Blue**: `#007AFF`
*   **Glass Light**: `rgba(255, 255, 255, 0.7)` with `backdrop-filter: blur(20px)`
*   **Glass Dark**: `rgba(0, 0, 0, 0.7)` with `backdrop-filter: blur(20px)`
